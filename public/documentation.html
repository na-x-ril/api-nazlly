API Documentation

TikTok API Endpoints Reference

Video Endpoints

Get Video Info:

/api/tt/v-get?url={videoUrl}

Required: url

Download Video:

/api/tt/v-download?url={videoUrl}&quality={hd}

Required: url | Optional: quality (hd)

Search Videos:

/api/tt/v-search?keywords={searchQuery}&unique_id={username}&days={number}&cursor={number}&count={number}

Required: keywords | Optional: unique_id, days, cursor (default: 0), count (default: 30)

Get Video Comments:

/api/tt/v-comments?url={videoUrl}&text={text}&days={number}&unique_id={username}&fuzzy={true}

Required: url | Optional: text, days, unique_id, fuzzy

Get Trending Videos:

/api/tt/v-trending?region={region}&days={number}

Required: region | Optional: days

Audio Endpoints

Get Audio Details:

/api/tt/audio-detail?url={videoUrl}

Required: url

Download Audio:

/api/tt/audio-download?url={videoUrl}

Required: url

User Endpoints

Get User Info:

/api/tt/user-get?username={username}

Required: username

Get User Posts:

/api/tt/user-posts?username={username}&days={number}

Required: username | Optional: days

Get User Favorites:

/api/tt/user-favorites?username={username}

Required: username

Get User Following:

/api/tt/user-following?username={username}

Required: username
